# Phishing URL Detector

## Deskripsi
Tool sederhana untuk mendeteksi URL phishing berdasarkan pola yang sering digunakan penyerang.

## Deteksi
- Penggunaan simbol @
- Kata mencurigakan (login, verify, secure)
- IP address langsung
- Struktur URL aneh

## Cara Menjalankan
python detector.py

## Catatan
Tool ini dibuat untuk pembelajaran cybersecurity dan tidak 100% akurat.
